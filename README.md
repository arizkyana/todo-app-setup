# Todo-app
Sample Angular Application

Requirement : 

1. NodeJS [docs here](http://www.nodejs.org)
2. Bower [docs here](http://www.bower.io)

Step : 

1. npm install
2. cd app && bower install
3. npm start

Running on : [http://localhost:3000/](http://localhost:3000/)
